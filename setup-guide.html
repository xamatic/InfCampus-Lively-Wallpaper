GradesWPInfCampus Setup

Use the wizard first, then finish the last two manual clicks here.

The installer can handle Node.js, Lively Wallpaper, server startup, dist sync, and autostart. Chromium still blocks silent unpacked extension installs, so use the image prompts below for the final steps.

If automatic Lively import worked, you only need to load the extension and sign in. If it did not, follow Step 3 exactly.
1

Run the installer

Double-click setup.bat. Recommended answers:

  • Install Node.js if missing: Yes
  • Install Lively Wallpaper: Yes
  • Auto-start server on login: your preference
  • Start server now: Yes
  • Try automatic Lively wallpaper import: Yes
2

Load the unpacked extension

Open chrome://extensions or edge://extensions, turn on Developer Mode, then click Load unpacked and select the grades-extension folder.

This step cannot be fully automated by the installer because Chromium blocks silent unpacked extension installs.
3

If needed, add the wallpaper in Lively

If the installer did not auto-apply the wallpaper, open Lively Wallpaper and add sites/wallpaper/wallpaper.html manually.

  • Open Lively
  • Choose Add Wallpaper
  • Pick sites/wallpaper/wallpaper.html
  • Apply it to your screen
4

Sign in and let the data sync

Use the extension popup or the login tab it opens, sign in to Infinite Campus, then wait a moment for grades and assignments to sync to the local server.

The wallpaper server is local: http://localhost:3001 and ws://localhost:3002. If data is blank, make sure the server is running and the extension is loaded.
